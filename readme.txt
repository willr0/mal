INTEGER.MAL:

MAL program which prompts user for a positive decimal integer and prints the following:

a) Total number of ones in the right half of the integer’s binary representations 

b) Total number of ones in the right half of the integer’s binary representations 

c) Highest power of two which evenly divides the integer

d) Value of the largest digit in the integer’s decimal representation


WORD.MAL

MAL program which prompts user for a line of text and prints the following:

a) Number of non-whitespace characters

b) Number of words

c) Maximum length of a word

d) Minimum length of a word

e) Max word

f) Min word
